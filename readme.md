## About the talk

Not a tools tour. Not a prompting demo.

A story about how product work changes when AI becomes part of the actual workflow — told through a year of real experiments, one sabbatical, and a part-time project building digital interventions for some of the most vulnerable people you can design for.

Honza covers the past year as a series of concrete situations: where AI stopped being a toy and became a practical collaborator. Strong emphasis on mindset — don't wait for a company AI strategy, try things in real work, map the limits, plug AI in where it actually helps. He closes with practical implications for designers, researchers, PMs, and developers.

**Full event description:** [UX Monday — How to build companies & products in 2026](https://www.meetup.com/asociace-ux/)
**Co-speaker:** Šimon Jůn (COO @ dotidot.io) — AI Agents in Startup Life

## Viewing the presentation

This presentation is written in **Marp** — Markdown for slides.

### In VS Code / Cursor
1. Install the [Marp for VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode) extension
2. Open `practical-ai-hotdog.md`
3. Click the Marp preview icon (top right) or `Cmd+Shift+P` → "Marp: Open Preview"
4. To present fullscreen: `Cmd+Shift+P` → "Marp: Export Slide Deck as HTML" → open in browser

### CLI export
```bash
npm install -g @marp-team/marp-cli
marp practical-ai-hotdog.md --html   # export as HTML
marp practical-ai-hotdog.md --pdf    # export as PDF
marp practical-ai-hotdog.md          # open as presentation
```
